## üíæ Where's My Data?

**Windows**: `C:\Users\YourName\.blackflag\`  
**Mac**: `/Users/YourName/.blackflag/`  
**Linux**: `/home/YourName/.blackflag/`

All data is stored as JSON files. Easy to backup, easy to inspect.

## üÜò Something Wrong?

### App won't start?
1. Check Node.js: `node --version`
2. Install dependencies: `npm install`
3. Check console output for errors

### Need to reset?
Delete the `~/.blackflag/` folder and restart the app.
